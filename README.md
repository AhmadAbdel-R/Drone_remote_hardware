# Drone Remote Hardware

Hardware workspace for my capstone remote + FCU board iterations.

## Repo Layout

- `hardware/fcu/FCU-hardware/`
  - FCU KiCad project and production artifacts
- `hardware/remote/FC_Remote-hardware/`
  - remote controller KiCad project and production artifacts
- `hardware/remote/FC_Remote-hardware/nrf_breakout/`
  - nRF breakout board subproject
- `mechanical/enclosure/`
  - enclosure and mechanical exports
- `mechanical/enclosure/remote_enclosure_stl/`
  - upload area for new remote enclosure STL revisions
- `mechanical/fcu/top_plate_updated_frame/`
  - upload area for updated FCU top plate/frame files
- `docs/`
  - repo organization notes

## Tools

- KiCad for schematic + PCB
- STEP/STL/3MF for mechanical handoff

## Practical Note

A lot of old backups and generated production files are intentionally kept for traceability between board revisions.
